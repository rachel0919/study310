제이쿼리 변경구문들

제이쿼리를 통해 html 또는 css를 변경하는 구문들

css()

제이쿼리에서 css의 설정을 바꾸는 구문

  • 체이닝기법 - css({'속성명':'속성값','속성명':'속성값'})
    체이닝기법은 복수형으로 css를 여러개 작성이 가능
  • 축약형단수 - css('속성명','속성값')

addClass() / removeClass()

html 태그에 class를 추가하거나 지울 때 사용

addClass('클래스명')
removeClass의 경우 ()를 비워놓을 경우 all의 의미를 갖고
하나씩 지울 경우 removeClass('클래스명') 으로 작성

text()

자바스크립트의 textContent의 의미로 text 요소의 내용을 삽입하거나 가져올때 사용한다

val()

자바스크립트의 value의 의미로 input value 값을 가져올 때 사용한다

attr()

html 태그의 속성의 값을 불러오거나 삽입할 때 사용

예로 img src="속성값"일 때 속성값을 부르는 방법으로
$('img').attr('src') 시 = 속성값을 변수로 지정할 수 있음

html()

html의 내부에 태그를 추가하는 구조
레이아웃의 스타일을 변경할 수 있음
