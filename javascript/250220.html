제이쿼리탐색구문

구문 설명
children() 자식요소선택 ()안에 이름으로 지칭 비워둘경우 모두 선택
parent() 부모선택요소. $(this)문과 함께 사용하는 경우가 많음 - 내가 선택한 요소의 부모다 $(this).parent()
siblings() 형제요소선택 : 나를 제외한 형제선택
find() 자손요소(후손) 선택
eq() 자손의 순번 선택자 : 0부터 시작
next() 선택자의 다음 요소
prev() 선택자의 이전 요소
not() 제외요소
구문 설명

prepend() 선택자 자식요소 중 처음 요소에 요소를 추가
after() 형제요소를 다음으로 추가
before() 형제요소를 앞전에 추가

이벤트구문

실제 사용자가 웹상에 특정한 행동을 취할 때 그에 대한 상호작용이 나타나는 형태

  • 클릭 - click(function(){실행문})
  • 마우스가 들어갔을 때 - mouseenter(function(){실행문})
  • 마우스가 나갔을 때 - mouseleave(function(){실행문})
  • 마우스가 움직였을 때 - mousemove(function(){실행문})
  • 브라우저의 스크롤이 움직였을 때 - $(window).scroll(function(){실행문})
  • 브라우저의 화면리사이즈가 되었을 때 - $(window).resize(function(){실행문})
