제이쿼리선택자

제이쿼리를 사용해서 선택자를 부르는 방법은 CSS 방법과 동일하되, $('선택자')로 작성한다.
클래스 - $('.클래스이름')
아이디 - $('#아이디이름')
태그명 - $('태그명')
자손자 의 경우 html과 CSS는 1 부터 시작이지만 , 자바스크립튼 0부터 시작됨
선택자:nth-child(1) - $('선택자:nth-child(1)')또는 - $('선택자').eq(0);
